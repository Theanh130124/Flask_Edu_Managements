<h1>Nếu là server trên pythonanywhere </h1> 
Không hỗ trợ upload trên Cloudinary 

- Cài thêm proxy
- pip install python-decouple
- Add thêm trong __init__
  + from decouple import config
  +  api_proxy = "http://proxy.server:3128"
